# Sakthi Sugars Chatbot

A chatbot developed using Python, Flask, SQLite, HTML, CSS, and JavaScript.

## Features

- FAQ-based responses
- Intent matching
- Keyword search
- Modern chat interface
- Animated UI
- Chat history
- SQLite database
- Responsive design

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## How to Run

```bash
pip install flask
python flask_app.py
```

Open:

```
http://127.0.0.1:5000
```
